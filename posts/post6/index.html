Bai 6: JavaScript - Hàm trong JavaScript

tháng 10 18, 2025
Programming
Javascript • Function

1. Hàm là gì? Link to heading

Hàm là một hoặc nhiều đoạn mã được viết ra để thực thi một hoặc nhiều hành động khi gọi nó. Hàm có khả năng gọi đi gọi lại được. Ví dụ thay vì tính tích của 2 số nhất định, chúng ta có thể sử dụng hàm để tính tích của 2 số bất kỳ mà chúng ta muốn.

2. Cấu trúc của hàm Link to heading

Hàm trong JavaScript có cú pháp tương tự như các ngôn ngữ khác, bắt đầu với từ khóa function. Có các dạng sau:

Hàm cơ bản Link to heading 

function funName() {
  //code
}

Ví dụ:

function getWebsite() {
  document.write("http://toidicode.com");
}

Hàm có tham số Link to heading 

function funName(param_1, ..., param_n) {
        //code
}

Ví dụ:

function getSum(a, b) {
  document.write("Tổng: " + (a + b));
}

Hàm có tham số mặc định Link to heading 

function funName(param_1 = value_1, ..., param_n = value_n) {
        //code
}

Ví dụ:

function getSum(a = 1, b = 2) {
  document.write("Tổng: " + (a + b));
}

3. Gọi hàm Link to heading

Sau khi tạo hàm, chúng ta có thể gọi hàm bằng các cách sau:

Gọi hàm cơ bản Link to heading 

funName();

Gọi hàm có tham số Link to heading 

funName(param_1, ..., param_n);

Gọi hàm có giá trị trả về Link to heading 

let result = funName();

4. Các ràng buộc đặt tên hàm Link to heading

  • Tên hàm phải bắt đầu bằng chữ cái (a-z, A-Z) hoặc dấu gạch dưới (_)
  • Không được bắt đầu bằng số hoặc các ký tự đặc biệt khác
  • Nên đặt tên có ý nghĩa và mô tả chức năng của hàm

5. Kết luận Link to heading

Hàm là một khái niệm cơ bản và quan trọng trong JavaScript. Việc sử dụng hàm giúp code trở nên có cấu trúc, dễ tái sử dụng và bảo trì hơn.
