Bài 2: Toán tử trong Javascript

tháng 10 18, 2025
Javascript • Học Lập Trình • Cơ Bản

Toán tử trong JavaScript Link to heading

Bài viết này tóm tắt các loại toán tử cơ bản trong JavaScript: toán tử gán, số học, kết hợp, quan hệ và logic, kèm ví dụ.

1. Toán tử gán Link to heading

Toán tử gán dùng để gán giá trị cho biến. Dấu gán là =.

Ví dụ:

var a = 1996;

2. Toán tử số học Link to heading

Các phép toán cơ bản: cộng, trừ, nhân, chia, chia lấy dư.

  • Cộng:
    • Nếu cả hai là số: cộng số.
    • Nếu một hoặc cả hai là chuỗi: nối chuỗi.
var a = 6;
var b = 5;
var c = "vu thanh ";
var d = "tai";

document.write(a + b); // 11
document.write("<br/>");
document.write(c + d); // 'vu thanh tai'
  • Trừ:
document.write(6 - 5); // 1
  • Nhân:
document.write(6 * 5); // 30
  • Chia:
document.write(10 / 5); // 2
  • Chia lấy dư (modulo):
document.write(6 % 5); // 1

3. Toán tử kết hợp (rút gọn) Link to heading

Các toán tử rút gọn giúp viết ngắn các phép gán kèm toán học.

Ví dụ Tương đương
a++ a = a + 1
a-- a = a - 1
a *= b a = a * b
a /= b a = a / b
a += b a = a + b
a -= b a = a - b
a %= b a = a % b

4. Toán tử quan hệ (So sánh) Link to heading

Trả về giá trị boolean (true hoặc false).

Toán tử Nghĩa
> Lớn hơn
>= Lớn hơn hoặc bằng
< Nhỏ hơn
<= Nhỏ hơn hoặc bằng
== Bằng (so sánh giá trị, không xét kiểu)
!= Khác

Lưu ý: các toán tử nhiều ký tự như >=, <= phải viết liền, không có khoảng trắng ở giữa.

(Trong thực tế còn có === và !== để so sánh chặt chẽ cả giá trị và kiểu.)

5. Toán tử logic Link to heading

Dùng để kết hợp các biểu thức boolean.

Toán tử Ý nghĩa Ví dụ Kết quả
&& Và — true nếu cả hai vế true true && false false
` ` Hoặc — true nếu ít nhất một vế true `true false` true
! Phủ định !false true

6. Lời kết Link to heading

Đã giới thiệu các toán tử cơ bản thường dùng trong JavaScript. Ôn lại và thực hành các ví dụ trên; bài tiếp theo sẽ tìm hiểu câu lệnh rẽ nhánh (if / switch).
