Bài 4: Vòng lập trong javascript

tháng 10 18, 2025
Javascript • Học Lập Trình • Cơ Bản

Vòng lặp trong JavaScript Link to heading

Một vòng lặp là chuỗi hành động được lặp đi lặp lại theo một nguyên tắc nhất định (ví dụ: sinh → lão → bệnh → tử). Trong JavaScript có các dạng vòng lặp chính: for, while, và do...while (và forEach, sẽ đề cập sau).

1. Vòng lặp là gì Link to heading

Vòng lặp cho phép thực thi lặp một đoạn mã nhiều lần cho tới khi đạt được điều kiện dừng.

2. Vòng lặp for Link to heading

Cú pháp chung:

for (khởiTạo; điềuKiện; bướcNhảy) {
  // code...
}
  • khởiTạo: giá trị ban đầu (ví dụ var i = 0)
  • điềuKiện: điều kiện để tiếp tục lặp
  • bướcNhảy: cập nhật sau mỗi lần lặp

Ví dụ in từ 1 đến 10:

for (var i = 1; i <= 10; i++) {
  console.log("Dòng số: " + i);
}

Vòng lặp for vô tận (bỏ trống điều kiện):

for (var i = 0; ; i++) {
  // code chạy mãi
}

3. Vòng lặp do…while Link to heading

Đặc điểm: thực thi thân vòng lặp trước, sau đó mới kiểm tra điều kiện (ít nhất chạy 1 lần).

Cú pháp:

do {
  // code
} while (condition);

Ví dụ minh họa (chạy 1 lần dù điều kiện false):

do {
  console.log(1);
} while (false);

In từ 1 đến 10 bằng do...while:

var i = 1;
do {
  console.log(i);
  i++;
} while (i <= 10);

4. Vòng lặp while Link to heading

Khác với do...while, while kiểm tra điều kiện trước khi thực thi thân vòng.

Cú pháp:

while (condition) {
  // code
}

In từ 1 đến 10 bằng while:

var i = 1;
while (i <= 10) {
  console.log(i);
  i++;
}

5. Vòng lặp lồng nhau Link to heading

Bạn có thể lồng vòng lặp bên trong nhau như các ngôn ngữ khác.

Vòng lặp for lồng nhau:

for (var i = 0; i <= 10; i++) {
  for (var j = i; j <= 10; j++) {
    process.stdout.write("*"); // hoặc console.log theo mục đích
  }
  console.log("");
}

Vòng lặp do...while lồng nhau:

var i = 0;
do {
  var j = i;
  do {
    process.stdout.write("*");
    j++;
  } while (j <= 10);
  console.log("");
  i++;
} while (i <= 10);

Vòng lặp while lồng nhau:

var i = 0;
while (i <= 10) {
  var j = i;
  while (j <= 10) {
    process.stdout.write("*");
    j++;
  }
  console.log("");
  i++;
}

6. Kết luận Link to heading
