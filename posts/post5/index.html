Bài 5: Break và Continue trong JavaScript

tháng 10 18, 2025
Javascript • Vòng-Lặp

1. Break Link to heading

Lệnh break trong JavaScript dừng toàn bộ vòng lặp hiện tại ngay lập tức.

Ví dụ: dùng break để dừng vòng lặp khi i == 5.

// khởi tạo vòng lặp chạy từ 0 đến 10
for (var i = 0; i <= 10; i++) {
  // in ra i hiện tại
  document.write(i + "<br>");
  // nếu i = 5 thì dừng vòng lặp
  if (i == 5) {
    break;
  }
}

Xem kết quả: sẽ in 0,1,2,3,4,5 rồi dừng.

2. Continue Link to heading

Lệnh continue bỏ qua phần còn lại của lần lặp hiện tại và chuyển sang lần lặp kế tiếp.

Ví dụ: bỏ qua khi i == 5.

// khởi tạo vòng lặp chạy từ 0 đến 10
for (var i = 0; i <= 10; i++) {
  // nếu i = 5 thì bỏ qua vòng lặp này
  if (i == 5) {
    continue;
  }
  // in ra i hiện tại
  document.write(i + "<br>");
}

Xem kết quả: sẽ in 0,1,2,3,4,6,7,8,9,10 (bỏ qua 5).

3. Lời kết Link to heading

Sau phần này bạn đã hiểu cơ bản cách sử dụng break và continue trong JavaScript. Ở bài tiếp theo chúng ta sẽ tìm hiểu về hàm (functions) trong JavaScript.
