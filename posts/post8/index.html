Bài 8: Mảng trong JavaScript — Khai báo, truy xuất và các hàm thường dùng

tháng 10 18, 2025
Javascript • Array • Tutorial
  1. Khai báo mảng

Trong JavaScript có 2 cách khai báo mảng phổ biến:

Khai báo bằng dấu []

// Cú pháp:
var arr = [value1, value2, ..., valueN];

// Ví dụ:
var arr = [1, 2, 4, 5, 9, 6];

Khai báo bằng new Array()

// Cú pháp:
var arr = new Array(value1, value2, ..., valueN);

// Ví dụ:
var arr = new Array(1, 2, 4, 5, 9, 6);
  1. Truy xuất mảng

Lấy phần tử theo chỉ số (index bắt đầu từ 0):

var arr = [1, 2, 4, 5, 9, 6];
console.log(arr[1]); // 2
  1. Các hàm/thuộc tính xử lý mảng

  • length

    • Thuộc tính trả về số phần tử:
    arr.length;

  • join()

    • Nối các phần tử thành chuỗi, tham số là ký tự phân cách:
    arr.join(); // "1,2,4,5,9,6"
arr.join("-"); // "1-2-4-5-9-6"

  • valueOf()

    • Trả về giá trị mảng (thường tương tự join với dấu phẩy khi ép chuỗi).

  • push(…)

    • Thêm 1 hoặc nhiều phần tử vào cuối mảng:
    arr.push(7);
arr.push(3, 8);

  • pop()

    • Xóa và trả về phần tử cuối cùng:
    arr.pop();

  • unshift(…)

    • Thêm 1 hoặc nhiều phần tử vào đầu mảng:
    arr.unshift(7);
arr.unshift(3, 8);

  • shift()

    • Xóa và trả về phần tử đầu tiên:
    arr.shift();

  • splice(index, howMany, …items)

    • Chèn/xóa phần tử tại vị trí xác định:
    // tại vị trí 1, không xóa, chèn 3 và 8
arr.splice(1, 0, 3, 8);
// để howMany > 0 sẽ xóa howMany phần tử bắt đầu từ index

  • concat(…)

    • Nối hai hoặc nhiều mảng, trả về mảng mới:
    var arr1 = [1, 2, 4, 5, 9, 6];
var arr2 = [3, 8, 7];
var merged = arr1.concat(arr2); // [1,2,4,5,9,6,3,8,7]

  • slice(begin, end)

    • Lấy một phần mảng (không thay đổi mảng gốc). end là chỉ số kết thúc (không bao gồm):
    arr.slice(3); // phần tử từ chỉ số 3 đến cuối
arr.slice(1, 4); // phần tử chỉ số 1,2,3

  • sort()

    • Sắp xếp mảng. Mặc định sắp xếp theo chuỗi; cho số nên dùng comparator:
    arr.sort(); // với số có thể cần: arr.sort((a,b)=>a-b)

  • reverse()

    • Đảo ngược thứ tự phần tử:
    arr.reverse();
  1. Kết luận
