Bài 7: Hàm alert(), confirm() và prompt() trong JavaScript

tháng 10 18, 2025
Javascript • Trình Duyệt • Lập Trình • Giao Diện

Trong JavaScript có ba hàm sẵn có dùng để hiển thị thông báo và tương tác nhanh với người dùng: alert(), confirm() và prompt(). Bài này giới thiệu cú pháp, cách dùng và ví dụ minh họa.

1. alert() Link to heading

Hàm alert() hiển thị một hộp thoại đơn giản kèm nội dung do bạn cung cấp. Hộp thoại chỉ có một nút (thường là “OK”).

Cú pháp:

alert(content);

Ví dụ:

alert("Học lập trình online toidicode.com");

Ghi chú: alert trả về undefined và dùng để thông báo, không để lấy dữ liệu từ người dùng.

2. confirm() Link to heading

Hàm confirm() hiển thị hộp thoại có hai nút (OK và Cancel). Nếu người dùng bấm OK thì hàm trả về true, nếu bấm Cancel thì trả về false.

Cú pháp:

const result = confirm(content);

Ví dụ:

if (confirm("Bạn có phải là fan của toidicode.com không?")) {
  document.write("Cảm ơn bạn rất nhiều!");
} else {
  document.write("Sao thế bạn?");
}

Ghi chú: confirm thích hợp để hỏi người dùng có đồng ý thực hiện hành động hay không (ví dụ xóa dữ liệu).

3. prompt() Link to heading

Hàm prompt() hiển thị hộp thoại kèm ô nhập liệu. Nếu người dùng nhập và bấm OK, hàm trả về chuỗi đã nhập; nếu bấm Cancel thì trả về null.

Cú pháp:

const result = prompt(content, defaultValue);

Ví dụ:

const age = prompt("Chào bạn, năm nay bạn bao nhiêu tuổi?");
if (age !== null) {
  document.write("Bạn đã nhập: " + age);
} else {
  document.write("Sao không trả lời à bạn?");
}

Ghi chú: prompt trả về chuỗi; nếu cần số, convert bằng Number() hoặc parseInt().

Ví dụ HTML để thử nhanh Link to heading 

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>Test Dialogs</title>
  </head>
  <body>
    <button onclick="alert('Xin chào!')">Test alert</button>
    <button
      onclick="if(confirm('Bạn muốn tiếp tục?')) alert('Đã chọn OK'); else alert('Đã chọn Cancel')"
    >
      Test confirm
    </button>
    <button
      onclick="const r = prompt('Tên bạn là gì?'); if (r!==null) alert('Chào ' + r); else alert('Không nhập')"
    >
      Test prompt
    </button>
  </body>
</html>

Kết luận Link to heading

Ba hàm alert, confirm, prompt rất tiện cho thao tác nhanh khi học hoặc debug. Trong ứng dụng thực tế, người ta thường dùng các thành phần UI tùy chỉnh (modal) để có trải nghiệm tốt hơn và kiểm soát giao diện.
