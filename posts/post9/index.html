DOM là gì và cách truy xuất DOM elements trong JavaScript Link to heading

1. DOM là gì? Link to heading

DOM = Document Object Model — mô hình đối tượng tài liệu.
DOM cho phép truy xuất và thao tác các thẻ HTML, thay đổi cấu trúc, nội dung hoặc thuộc tính của trang bằng JavaScript.

2. Các cách truy xuất phần tử DOM Link to heading

Truy xuất theo id Link to heading

Sử dụng document.getElementById(‘idName’).

Ví dụ:

<div id="toidicode">TOIDICODE.COM</div>

<script>
  var element = document.getElementById("toidicode");
  var content = element.innerHTML;
  alert(content);
</script>

Truy xuất theo class Link to heading

Sử dụng document.getElementsByClassName(‘className’) — trả về HTMLCollection (mảng-like).

Ví dụ:

<div class="hello">Chào mừng các bạn</div>

<script>
  var elements = document.getElementsByClassName("hello");
  // phần tử đầu tiên
  alert(elements[0].innerHTML);
</script>

Truy xuất theo tag name Link to heading

Sử dụng document.getElementsByTagName(’tagName’) — trả về tất cả thẻ cùng loại.

Ví dụ:

<div>...</div>
<div>...</div>

<script>
  var divs = document.getElementsByTagName("div");
  alert(divs[0].innerHTML);
</script>

Chú ý: getElementsByClassName và getElementsByTagName trả về tập hợp phần tử, có thể duyệt bằng index hoặc vòng lặp.

Tìm phần tử con bên trong phần tử khác Link to heading

  • Cách 1: Lồng các getElement… vào nhau
<div class="parent">
  <p class="child">Nội dung</p>
</div>

<script>
  var parent = document.getElementsByClassName("parent");
  var child = parent[0].getElementsByClassName("child");
  alert(child[0].innerText);
</script>
  • Cách 2: Sử dụng querySelectorAll với cú pháp kiểu CSS
<div id="hello">Chào mừng</div>

<script>
  var el = document.querySelectorAll("div#hello");
  alert(el[0].innerText);
</script>

querySelector/querySelectorAll hỗ trợ selector giống CSS và rất linh hoạt.

3. Kết luận Link to heading

  • Các API cơ bản để truy xuất DOM: getElementById, getElementsByClassName, getElementsByTagName, querySelector(All).
  • Lưu ý tập hợp trả về có thể có nhiều phần tử — cần dùng index hoặc duyệt.
  • Trong ví dụ trên đã dùng innerHTML/innerText; phần sau sẽ giải thích chi tiết các thuộc tính này.
